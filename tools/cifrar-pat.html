Cifrar PAT · RSB Eventos

Esta página corre 100% no browser. Nada é enviado para nenhum servidor. O PAT cifrado pode ser colado no código-fonte das apps (SPA Presencas e Eventos).

Como usar:
  1. Cola o PAT do GitHub no campo abaixo.
  2. Introduz a password admin (a mesma que se usa para autorizar acções no admin).
  3. Clica Cifrar.
  4. Copia a string de saída e cola no código nas duas apps (constante PAT_CIFRADO).
  5. Faz commit + push. Pronto.
